# Cx2bro-Extensions
[Cx2bro](https://github.com/zeli624233/Cx2bro)项目支持的游戏列表。
- 本项目的游戏很多都是**2022年11月以后发售**的游戏，如果你喜欢的游戏，不是这个时期的，这个项目并不能帮到你。
- 个人精力有限，没办法支持那么多的游戏，如果你已经制作了列表以外的扩展集的话，并且成功率（>83%）（要有截图）,可以先fork本项目，然后把你的文件上传到Extensions文件夹（符合规范要求。），再把下面的表格也填了（除了备注宽松点，其他的要按真实的填（体积是两个int文件大小的相加，发售日这些可以去2dfan查，入库编号，看下面，如果最大是06，你就填07）），然后Pull request 请求就行。我这边会审核，通过了，就合并。
